QR GENERATOR

Buat QR Code kustom secara instan dan mudah. Dukung penambahan logo, ubah warna, serta sesuaikan bentuk titik sesuai keinginan Anda.

Panduan & Informasi

1. Cara Membuat QR Code

Centang persetujuan di halaman awal, lalu klik tombol "Buat QR Code Baru" untuk membuka form pembuatan.

2. Memasukkan Data

Ketik atau tempel alamat URL website, teks, atau nomor pada kolom tautan yang tersedia.

3. Kustomisasi Kreatif

  • Logo: Pilih file gambar untuk menyisipkan logo di tengah.
  • Warna: Pilih warna bintik utama dan warna dasar background.
  • Bentuk: Ganti model titik (dots) dan bentuk siku penanda QR.

4. Ambil Hasil

Ketuk tombol "Perbarui Preview" untuk meninjau, lalu tekan "Download QR Code" untuk menyimpan gambar PNG beresolusi tinggi.
